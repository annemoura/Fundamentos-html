Semântica no HTML

Uso de tags com significado para melhorar estrutura, organização e acessibilidade.

O que é semântica?

Semântica no HTML significa usar elementos como <header>, <main>, <section> e <footer>.

Por que usar?

Ajuda na organização do código, melhora SEO e acessibilidade.
